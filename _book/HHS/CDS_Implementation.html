  • 确定可以使用CDS的工作流中的点 > CDS designers can use the taxonomy to identify points in the workflow when CDS can be used and design a CDS tool to fit that context. Designers can then tag the CDS tools with terms from the taxonomy to inform practices about the intended use of the CDS. CDS repositories can create tags based on the taxonomy to enable workflow-related organization and searches. CDS implementers can refer to the taxonomy when developing maps of the workflows and can use the terms to improve communication with their CDS vendor.
